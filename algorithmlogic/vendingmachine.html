Máquina de Vendas

- Projeto -

O Objetivo deste projeto era criar em python uma máquina de vendas com estoque finito e que calcule com quais cédulas/moedas devolver o troco ao usuário, com desafios adicionais para adicionar um modo de edição dinâmico e troco finito.

Eu acabei por fazer este projeto utilizando JSON para manter data persistente como diferencial dos outros projetos, criando um script em python que controlava todas minhas operações de dados como: Edição, Remoção e Adição de Produtos. Enquanto outro script em python controlava a interface e lógica da própria maquina, chamando funções do controlador de dados quando necessario
