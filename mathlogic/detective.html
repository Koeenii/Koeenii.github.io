- Projeto -

O Objetivo deste projeto era criar um programa quer fornece uma narrativa de misterio, dando hipoteses para que o jogador resolvesse mistérios através de dução lógica, sendo compravado com as regras de indução.

O Diferencial do meu projeto foi sua estruturação, eu projetei ele com a facilidade de adição de novos casos em mente, sendo apenas necessario fornecer os dados do mistério, sem ter de programar nada para adicionar um mistério novo. O Trabalho foi entregue com 2 mistérios prontos, um de minha autoria e outro era o mistério que a professora havia usado como exemplo, porém implementado para meu sistema.
